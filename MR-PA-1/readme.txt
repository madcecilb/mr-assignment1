command to run program:
	java -jar <MR.Assignment-2-4.jar location> <folder to be indexed and searched> <query>
for example: 
	java -jar C:\MR.Assignment-2-4.jar C:\data function
by default program searches for a query: 
	function
so, command:
	java -jar C:\MR.Assignment-2-4.jar C:\data
searches for:
	function 
	in a folder C:\data
	
file locations:
	MR.Assignment-2-4.jar inside folder /runnable_jar
	documentation inside folder /doc
	data for search inside folder /data
